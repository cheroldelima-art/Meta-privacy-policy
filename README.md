# Privacy Policy for AI Agent Gestion Pages

This privacy policy governs your use of the application **AI Agent Gestion Pages** created by cheroldelima-art.

### 1. Information Collection and Use
Our application does not collect, store, or share any personal data from its users. If the application processes any data through Meta APIs, it is processed locally or in memory solely to provide the intended functionality (managing pages) and is never saved on external servers.

### 2. Data Deletion
Since we do not collect any user data, there is no user data to delete. If you wish to revoke the application's access to your Meta account, you can do so directly via your Meta User Settings.
